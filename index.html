Conectar fontes

Conecte o Trakt para filmes/séries e/ou o AniList para animes. Pode usar os dois ao mesmo tempo.

ℹ️ Como fazer login (Ajuda / Passo a Passo)
● Trakt (OAuth Automático)
  1. Cole seu Client ID do Trakt (link "Criar app" abaixo ajuda a obter um).
  2. Configure a Redirect URI no site do Trakt como: urn:ietf:wg:oauth:2.0:oob
  3. Clique em "Conectar Trakt via OAuth" e siga as instruções inserindo o código exibido na tela.
● AniList (OAuth Automático - com Servidor)
  1. Se estiver rodando a página em um servidor local (ex: extension Live Server do VS Code, rodando em http://127.0.0.1:5500/...).
  2. No desenvolvedor do AniList, defina o Redirect URI para a URL exata do seu navegador (ex: http://127.0.0.1:5500/trakt-site.html).
  3. Insira o Client ID e clique em "Conectar AniList via OAuth".
● AniList (Token Manual - Recomendado para arquivo local file:///)
  1. Se você abriu o arquivo direto com 2 cliques no Windows.
  2. No desenvolvedor do AniList, crie um client e defina o Redirect URI como exatamente: https://anilist.co/api/v2/oauth/pin
  3. Abra esta URL no seu navegador para autorizar (o ID abaixo muda ao digitar no campo "Client ID" do AniList):
    Abrir link de autorização (com ID 44001)
  4. Copie o token de acesso exibido na tela após autorizar, e cole-o no campo Access Token manual do AniList!
